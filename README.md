# Сайт кинотетатра 

## Стэк технологий:
-  react
-  redux-toolkit
-  redux-thunk
-  react-hook-form
-  eslint, prettier, husky

## Функционал:
- авторизация пользователя
- личный кабинет
- афиша фильмов
- информация о фильме, расписание
- возможность заказать билеты
- имитация опт кода для регистрации
